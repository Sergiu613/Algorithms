# Insertion Sort Collection

This repository contains implementations of the **Insertion Sort** algorithm in several programming languages and architectures, ranging from high-level languages to assembly.

## Algorithm Description

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time. It is much like the way you sort playing cards in your hands: you pick one card and "insert" it into its correct position relative to the cards you already hold. It is efficient for small data sets or nearly sorted arrays.

## Complexity

* **Best Case:** $O(n)$ (when the array is already sorted)
* **Average Case:** $O(n^2)$
* **Worst Case:** $O(n^2)$
* **Space Complexity:** $O(1)$ (In-place)
